# Serial Killers EDA

> ⚠️ **Work in Progress** — this project is actively being developed.

## Overview
An exploratory data analysis of serial killer datasets sourced from Kaggle. 
This project investigates victim counts, geographic patterns, and active 
periods using R.

## Methods
- Exploratory Data Analysis (EDA)
- Statistical Modelling & Inference
- Data Visualisation

## Repository Structure
- `data/` — raw and processed datasets
- `scripts/` — R scripts for analysis
- `outputs/` — plots and results

Part of an ongoing data analytics portfolio.
